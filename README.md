# Expo Image Picker and Gallery Example

This is a simple example of how to use Expo's Image Picker and Gallery features in a React Native application. The app
allows users to select images from their device's gallery or take new photos using the camera. It also displays the
selected images in a grid format.

## Features
- Select images from the device's gallery
- Take new photos using the camera
- Display selected images in a grid layout
- Remove selected images using a button
